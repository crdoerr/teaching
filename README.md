# Teaching
This MATLAB code is for simulating optical waveguide for photonic integrated ciruits.  It was developed by Chris Doerr for the purpose of enabling people to simulate and explore optical waveguides without the need to purchase software other than MATLAB.  It is also to instruct how optical simulation code works.  The code is written as simple and compact as possible.

Please feel free to contribute to it.

The mode calculator needs PMLs and complex fields can be added so that loss can be calculated. Also, a variable mesh would be helpful.

The FDTD calculator needs many improvements.  One immediate improvement is to code the engine in C to make it faster. It also needs PMLs and variable mesh.
